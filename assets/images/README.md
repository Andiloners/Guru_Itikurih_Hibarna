# Folder Logo

Silakan tempatkan file logo sekolah di folder ini dengan nama:
- **logo.png** (format PNG dengan transparansi)
- atau **logo.jpg** (format JPG)

## Ukuran yang Disarankan:
- Untuk dashboard: minimal 70x70px
- Untuk login: minimal 120x120px
- Format: PNG dengan background transparan (disarankan)

## Cara Menggunakan:
1. Simpan file logo dengan nama `logo.png` di folder ini
2. Jika menggunakan format JPG, ubah nama file menjadi `logo.jpg` dan update path di:
   - `index.php` (baris ~123)
   - `login.php` (baris ~26)

## Catatan:
- Jika logo tidak ditemukan, sistem akan menampilkan icon emoji sebagai fallback
- Pastikan logo memiliki kualitas yang baik untuk tampilan yang optimal
